Equiple Guide

    Equiple - item guessing game based off Wordle. You have to guess equipment from the game Realm of the Mad God in 10 tries. Hints for you are six item attributes, which are:
  • Tier - item tier, can be either "UT" (Untiered), "ST" (Set Tiered) or "Tiered". Reskinned items marked as "Reskin" afterwards, which makes 6 tiers in total. If target item is reskin, you will have to guess the reskin.
    Note: Antinomy Gear, despite having wc tops stats are not considered as reskins in the game, and neither are here.
  • Type - slot type, either "Weapon", "Ability", "Armor" or "Ring".
  • Color - dominant color of the item. The color is defined automatically so it's not always what you might expect it to be. Keep that in mind and always check the color in search results to avoid mistakes. Colortip (background) will change depending how far from the target you are.

    You can hover over color to see it's exact name. Use this to compare it with search results. First goes the "exact" color of the item, then in square brackets closest color group. You can use that to determine whether or not two colors are similiar.
  • Xp Bonus (Fame Bonus) - xp bonus given by the item.
  • Feedpower - feedpower of the item.
There are two gamemodes: Daily and Normal.
- In Daily everyone is given the same item, and you can only play it only once per day. If you forget to finish your daily game, it will expire at midnight UTC and this counts as a lose, so you will lose your streak.
- In Normal you can play as much as you want and you can exclude reskinned items from the game. Tick off "Exclude reskins" checkbox before starting a new game.
Consecutive wins in both gamemodes increase your respective winning streak, losing on the other hand voids it. Best players show up in the Leaderboard.
  • If you search up specific item type, you will be given all items that match the search. For example if you search "Katana" the result will contain all katanas regardless if it states this in the name or not.
  • You can also look up specific tiers of items in search. "ut" will return you all UT items, "reskin" will return you all reskins, "t10" will give all t10 items, etc. You can use this for most attributes except feedpower and famebonus.
  • You can combine prompts for more specific search, e.g. "st weapon", "t6 ability" or"ut ring".
  • Try to remember unique item cases, for example The Forgotten Crown is the only UT ring with 8% xp bonus and 1400 feedpower; Corruption Tether is the only ST weapon with 7% xp bonus. There are more items like this! ;)

    Let's go over quick game example. First pick shouldn't matter that much, but can lean in you in the right direction. I think good strategy is to pick something average.
    My first guess is Sunset Tachi, and it was absolutely wrong. Howewer now we know that target is not a tiered item, which filters away a huge chunk of items (probably the biggest). But not only that, now we know the item is not a weapon and can not be a "regular"set piece. There are only two st items that have 8% xp bonus, if target item happened to be one of them we can quickly find out.
    My second guess is Ballistic Star, which was picked because of high feedpower and 8% xp bonus. If xp bonus wasn't right I'd immediately know what that item is (Chrysalis).
    And now we know that it is a blue-ish colored ring with feedpower value somewhere in between 1300 and 1100. There aren't that many rings that fit this description. I can say with confidence this is in fact The Twilight Gemstone!
    Yes, color wasn't that much help in this game, but that's expected. The color is there to help you in difficult situations that might arise while trying to guess more tightly grouped items, such as tiered rings, themed sets or reskins
    Game went from many design variations from initial idea. Tricky task was to pick attributes to base the game off. Number of shots and range seemed like a cool addition to the game until you had to guess something like tiered ring or st ring (Did you know there're 11 st rings with 4% xp bonus and 600 feedpower in the game?). Which turns into just blind guessing right after you get the tier right.
    The same but different issue comes from The Machine whites, there are 17 abilities that have exact same tier, lootbag, xp bonus, feedpower and slot type. 10 of them have no shots whatsoever, 7 have no range and 9 have no damage values. Which makes it harder to balance the game around if you had to use damage, range and/or amount of shots. Dungeon is also the same (if you had that kind of idea in your mind ;) ).
    Color initially was introduced because of this two problems, and so far it seems like the only good solution.
